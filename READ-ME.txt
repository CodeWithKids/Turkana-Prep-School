  # Turkana Preparatory School Website

Official website for Turkana Preparatory School, established in 2024, providing a modern and informative online presence for students, parents, and staff.

## Design & Features

- **Modern, Responsive Design**: Looks great on all devices
- **Clean Typography**: Montserrat font for all text with optimized readability
- **Color Scheme**:
  - Primary: Dark Green `#2c5234`
  - Secondary: Cream `#f5f5dc`
  - Accent: Gold `#f2b816`
- **Key Pages**:
  - Home
  - About Us
  - Classes
  - Teachers
  - Gallery
  - Events
  - Contact

## Setup & Installation

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   cd kindergarten-website-template-2
   ```

2. **Add Media Assets**
   - Place the school logo as `logo.png` in the `img/` folder
   - Add additional images to the `img/` folder as needed

3. **Run the Website**
   - Open any `.html` file in a web browser to view the site
   - For local development, use a local server (e.g., Live Server in VS Code)

## Customization

### Update Content
- Edit the HTML files directly to update text content
- Replace images in the `img/` folder (keep original filenames)

### Update Styling
- Main styles are in `css/style.css`
- Theme colors and custom styles are in `css/theme-colors.css`
- Fonts are loaded from Google Fonts (Montserrat)

### Scripts
- `update_theme.py`: Updates theme across all pages
- `add_back_to_top.py`: Adds back-to-top button functionality
- `update_fonts.py`: Updates font settings across the site
